# Neuroglancer Views

## Authentication

If this is your first time using MICrONS, you'll need to accept the 
[Terms of Service](https://global.daf-apis.com/sticky_auth/api/v1/tos/2/accept)
with a Google account to be able to view these links properly. You'll then be prompted to
sign in (make sure to use the same account) when opening one of the links below for the 
first time in a session.
